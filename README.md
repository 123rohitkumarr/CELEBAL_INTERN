## 📌 Dataset

Tesla EV Deliveries and Production Data (2015–2025)

Dataset Source:
https://www.kaggle.com/datasets/nalisha/tesla-ea-deliveries-and-production-data20152025

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
